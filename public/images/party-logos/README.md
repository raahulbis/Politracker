# Party Logos

This directory contains the official party logos as SVG files.

## Current Files

- `liberal.svg` - Liberal Party of Canada
- `conservative.svg` - Conservative Party of Canada  
- `ndp.svg` - New Democratic Party
- `bloc-quebecois.svg` - Bloc Québécois
- `green-party.svg` - Green Party of Canada

## Downloading Official Logos

The current files are placeholders. To get the official logos:

1. Visit the Wikimedia Commons pages:
   - Liberal: https://commons.wikimedia.org/wiki/File:Liberal_Party_of_Canada_Logo.svg
   - Conservative: https://commons.wikimedia.org/wiki/File:Conservative_Party_of_Canada_logo.svg
   - NDP: https://commons.wikimedia.org/wiki/File:New_Democratic_Party_of_Canada_logo.svg
   - Bloc Québécois: https://commons.wikimedia.org/wiki/File:Bloc_Qu%C3%A9b%C3%A9cois_logo.svg
   - Green Party: https://commons.wikimedia.org/wiki/File:Green_Party_of_Canada_logo.svg

2. On each page, click "Download" or "Original file" to get the SVG

3. Save the files with the names above in this directory

Alternatively, you can try running:
```bash
npm run download-logos
```

Note: The download script may need the actual file URLs from Wikimedia Commons, which can be found by inspecting the download link on each page.
